Student Designed, Hand Coded

This website was designed and coded by students from the robotics club. It is not a theme or template. Although it requires more effort, writing a website from scratch extends the web designers more creative power. The code is also optimized to run on the website as efficiently as possible.

Uses HTML5 and CSS3 Technologies

Our site utilizes the very latest in web technologies, including HTML5 and CSS3. We strive to ensure that our site remains up to date with the latest trends in design and layout. At the same time, we make sure that older web browsers remain compatible with our designs.

Follows the 960 Standard

Our website uses the 960px grid system from 960.gs to ensure a consistent and easily adjustable content display. All modern computer monitors are at least 1024px across, which means our website is viewable on almost any computer you can get your hands on.

Powered by Google App Engine

Our website is hosted as a Google App (Python). The Google Appengine allows us to rapidly develop new features on our website, launch multiple versions, and easily develop, deploy, and utilize databases.

Facebook Integration

Our site uses the Facebook Graph API to integrate with our Facebook page. Most importantly, we use this functionality to automatically update our gallery page. Photos from our Facebook albums automatically appear in the gallery.
